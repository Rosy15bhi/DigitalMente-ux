# DigitalMente — UX Project

Progetto di UX Design per il corso di **Progettazione Funzionale**.  
Rosalia Lazari · matr. 1034216 · A.A. 2025/2026

## Il Progetto

**DigitalMente** è una piattaforma web per la digital literacy degli anziani italiani (65+).  
Il progetto nasce dall'analisi del divario digitale in Italia: solo il 12% degli over 65 possiede competenze digitali di base.

---

## Metodologia

| Fase | Contenuto |
|------|-----------|
| Fase 0 | Brief, obiettivi, KPI, stakeholders, scope |

| Fase 2 | Problem statement, How Might We, requisiti, accessibilità |
